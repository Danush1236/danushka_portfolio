GO BUS Mobile App

June 2024

Project Overview

Designed a modern and user-friendly mobile application for GO BUS, a public transportation service. The app aims to simplify the bus travel experience by providing real-time bus tracking, digital ticketing, and seamless journey planning for commuters.

Key Features

🚌

Real-time Tracking

Live bus location tracking and arrival time predictions

🎫

Digital Ticketing

Purchase and store digital tickets with QR code scanning

🗺️

Journey Planner

Smart route planning with multiple transport options

🔔

Notifications

Real-time alerts for delays, cancellations, and updates

⭐

Favorites

Save frequent routes and stops for quick access

Design Process

👥

User Research

Conducted user interviews and analyzed commuter needs

✏️

Wireframing

Created low-fidelity prototypes for key user flows

🎨

UI Design

Developed a modern, accessible interface design

🧪

User Testing

Conducted usability testing and iterated on feedback

Challenge & Solution

Challenge

Creating an intuitive interface that simplifies complex public transportation information while maintaining accessibility for users of all ages and technical abilities.

Solution

Implemented a clean, hierarchical design with clear visual cues and minimal cognitive load. Used color coding and intuitive icons to make navigation straightforward, while ensuring all features are accessible through multiple interaction patterns.
