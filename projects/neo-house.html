NEO House

2024

Project Overview

NEO House is a modern restaurant website prototype designed using Figma. The project focuses on creating an intuitive and visually appealing interface that showcases the restaurant's menu, ambiance, and services. The design emphasizes user experience and accessibility while maintaining a contemporary aesthetic that reflects the restaurant's brand identity.

Design Process

🔍

Research

Studied popular restaurant websites to understand user expectations and industry standards

📝

Wireframing

Created low-fidelity wireframes to outline layout and user flow

🎨

UI Design

Developed high-fidelity, responsive design focusing on visual hierarchy and accessibility

✅

Testing & Feedback

Reviewed the design with peers and implemented improvements based on feedback

Challenge & Solution

Challenge

Creating an engaging and intuitive menu layout that would effectively showcase the restaurant's diverse offerings while maintaining visual appeal.

Solution

Implemented a categorized menu system with clear visual hierarchy, high-quality food photography, and interactive elements to enhance user engagement.
