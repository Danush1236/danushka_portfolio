IOT Smart Baby Room

October 2023

Project Overview

The Smart Baby Room is designed to help busy parents monitor and care for infants under 5 months old using IoT technology. It creates a safe, comfortable, and automated environment by integrating various sensors and systems to ensure optimal conditions for baby care—addressing issues such as safety, health, and convenience.

Key Features

🎵

Audio Monitoring

Detects baby cries and soothes with automated cradle movement

🌡️

Temperature & Humidity Control

Monitors and adjusts the room environment

💡

Lighting System

Automated lights based on brightness detection

🚪

Auto Door & Window

Hands-free operation for safety and convenience

🔔

Notification System

Alerts for comfort issues and diaper wetness

🔒

Security System

Real-time activity monitoring with alerts

🚨

Safety System

Detects fire or gas leaks and alerts parents

Technical Implementation

📡

Sensors Used

Ultrasonic, Gas, Humidity/Temperature, LDR, IR, Rain, and Microphone sensors

🎮

Controllers

Arduino board and NodeMCU for sensor integration and system control

⚙️

Actuators

Servo motors for door/window operations, LEDs for lighting, and buzzer for alerts

🌐

Connectivity

IoT-based real-time monitoring and notifications

Technologies Used

Arduino Uno NodeMCU Servo Motors Sensors Arduino IDE C/C++ IoT
