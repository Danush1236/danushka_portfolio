Apparel Printing Management System

May 2024

Project Overview

Developed a comprehensive web-based management system to digitize and streamline operations for Carlos Embellishers, a Sri Lankan apparel printing company. The system aims to replace manual processes, enhancing efficiency, accuracy, and real-time data access across departments.

Key Features

👥

Role-based Dashboards

Customized interfaces for CEO, System Admin, Production Manager, Marketing, and Stores

🤝

Customer & Supplier Management

Centralized databases for orders, quotations, and payments

📊

Production Tracking

Real-time monitoring of material requests, processes, and quality control

📈

Automated Reporting

Generate financial, sales, production, and inventory reports

🔒

Security

Secure login, password recovery, and role-based access control

🧪

🎨

Final Design

Implemented refined UI design based on feedback

Challenge & Solution

Challenge

Simplifying a complex ordering process with many customization options while maintaining efficiency and reducing errors.

Solution

Designed a step-by-step wizard interface with visual previews at each stage. Implemented a modular design system that allows for easy customization while maintaining consistency.
